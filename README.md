# NS Studio

High-fidelity UI/UX design mockup for a cutting-edge student web agency.
Features a dark mode aesthetic, deep charcoal background with a vibrant iridescent fluid 3D blob, and dynamic scroll-triggered animations.

## Tech Stack
- Vanilla HTML5
- Vanilla CSS3 (Custom Properties, Flexbox/Grid, Animations)
- Vanilla JavaScript (IntersectionObserver, DOM Manipulation)

## Local Development

The site is a pure static frontend. You can run it using any static file server.

If you have Python installed, you can use:
```bash
python3 -m http.server 8000
```
Then navigate to `http://localhost:8000`.

Alternatively, use a tool like VS Code Live Server or `npx serve`.
